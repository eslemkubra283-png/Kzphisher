# Kzphisher
Kzphisher tamamamen ücretsiz bir SMS bomber dir 
## 🚀 Kurulum ve Kullanım

Aracı Termux üzerinden çalıştırmak için aşağıdaki komutları sırayla kopyalayıp yapıştırın:

1. `pkg install git python -y`
2. `git clone https://github.com/eslemkubra283-png/Kzphisher.git`
3. `cd Kzphisher`
4. `pip install requests colorama`
5. `python kzphisher.py`
